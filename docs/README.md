# Documentación del proyecto (índice)

He organizado la documentación en la carpeta `docs/` para que sea más fácil navegar:

- docs/docker/README.md → Guía de Docker (producción y desarrollo)
- docs/seed/README.md → Guía para generación de datos (fake_data)
- docs/general/README.md → Índice y enlaces generales
- docs/archive/originals.md → Copia y lista de los archivos .md originales

Sugerencia rápida:
- Abre `docs/docker/README.md` para toda la información de despliegue y desarrollo.
- Abre `docs/seed/README.md` para ejecutar el generador de datos dentro de Docker.

Si quieres que mueva físicamente (elimine del root) los archivos originales, dímelo y lo hago.
